O Transportin´ é uma aplicação web desenvolvida com o objetivo de centralizar e facilitar o controle e a administração de serviços de transporte escolar. A plataforma oferece uma interface intuitiva para o gerenciamento de rotas, alunos, motoristas e veículos, promovendo maior organização, segurança e eficiência no dia a dia das instituições de ensino.

⚙️ Funcionalidades

📌 Cadastro e gerenciamento de alunos
🚐 Controle de veículos e frota
👤 Gestão de motoristas e responsáveis
🗺️ Definição e visualização de rotas
📊 Painel administrativo com visão geral do sistema
📱 Interface responsiva para diferentes dispositivos
