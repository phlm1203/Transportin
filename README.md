<h2> Sobre o Repositório</h2>

<p>O Transportin´ é uma aplicação web desenvolvida com o objetivo de centralizar e facilitar o controle e a administração de serviços de transporte escolar. A plataforma oferece uma interface intuitiva para o gerenciamento de rotas, alunos, motoristas e veículos, promovendo maior organização, segurança e eficiência no dia a dia das instituições de ensino.</p>

⚙️<h2> <b>Funcionalidades:</b></h2>
<br>
<ul>
<li>📌 Cadastro e gerenciamento de alunos</li>
<li>🚐 Controle de veículos e frota</li>
<li>👤 Gestão de motoristas e responsáveis</li>
<li>🗺️ Definição e visualização de rotas</li>
<li>📊 Painel administrativo com visão geral do sistema</li>
<li>📱 Interface responsiva para diferentes dispositivos</li>
</ul>

⚙️ <h2> Linguagens de Programação Utilizadas</h2>

  <li>CSS</li>
  <li>JavaScript</li>
</ul>
